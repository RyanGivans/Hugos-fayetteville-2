# Hugo’s Fayetteville Redesign Concept

GitHub Pages files:
- `index.html`
- `styles.css`
- `app.js`
- `menu.json`

## How to publish
1. Create a new GitHub repository.
2. Upload these four files to the root of the repository.
3. Go to Settings → Pages.
4. Under “Build and deployment,” choose Deploy from a branch.
5. Choose `main` and `/root`, then Save.
6. Your live preview will be at `https://YOURUSERNAME.github.io/REPOSITORY-NAME/`.

## Notes
This is a static concept site. The images are loaded from the current Hugo’s website URLs. For a final paid build, download and optimize those images, confirm rights/permission with the owner, and update current prices/hours before launch.
